# Linear Regression from Scratch

Verified, runnable code from the [Linear Regression from Scratch](https://dadops.dev/blog/linear-regression-from-scratch/) blog post.

## Scripts

- **simple_regression.py** — 1D linear regression with closed-form solution + sklearn validation
- **normal_equation.py** — Normal equation and pseudoinverse for multiple features
- **gradient_descent.py** — Gradient descent with and without feature scaling
- **polynomial_regression.py** — Polynomial regression and bias-variance tradeoff
- **ridge_lasso.py** — Ridge (L2) and Lasso (L1) regularization
- **evaluation.py** — R², adjusted R², RMSE, MAE, and residual diagnostics

## Run

```bash
pip install -r requirements.txt
python simple_regression.py
python normal_equation.py
python gradient_descent.py
python polynomial_regression.py
python ridge_lasso.py
python evaluation.py
```
