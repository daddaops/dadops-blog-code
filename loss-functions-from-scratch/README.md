# Loss Functions from Scratch

Verified, runnable code from the [Loss Functions from Scratch](https://dadops.dev/blog/loss-functions-from-scratch/) blog post.

## Scripts

- **mse_and_sigmoid.py** — MSE loss, gradients, sigmoid, and the vanishing gradient problem
- **bce_gradient.py** — Binary cross-entropy, maximum likelihood derivation, gradient comparison
- **information_theory.py** — Surprise, entropy, cross-entropy, KL divergence
- **softmax_ce.py** — Softmax, categorical cross-entropy, log-softmax stability
- **gradient_showdown.py** — MSE vs BCE gradient comparison table

## Run

```bash
pip install -r requirements.txt
python mse_and_sigmoid.py
python bce_gradient.py
python information_theory.py
python softmax_ce.py
python gradient_showdown.py
```
