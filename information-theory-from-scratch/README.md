# Information Theory from Scratch

Verified, runnable code from the [Information Theory from Scratch](https://dadops.dev/blog/information-theory-from-scratch/) blog post.

## Scripts

- **entropy.py** — Shannon entropy, information content, binary entropy curve
- **cross_entropy.py** — Cross-entropy, binary and categorical cross-entropy
- **kl_divergence.py** — KL divergence asymmetry, forward vs reverse KL on bimodal distributions
- **mutual_information.py** — MI-based feature selection on synthetic data
- **perplexity.py** — Perplexity from log-probs and cross-entropy loss
- **connections.py** — Unifying table of information-theoretic objectives in ML

## Run

```bash
pip install -r requirements.txt
python entropy.py
python cross_entropy.py
python kl_divergence.py
python mutual_information.py
python perplexity.py
python connections.py
```
