# Genetic Algorithms from Scratch

Verified, runnable code from the [Genetic Algorithms from Scratch](https://dadops.dev/blog/genetic-algorithms-from-scratch/) blog post.

## Scripts

- **basic_ga.py** — Binary-encoded GA for multimodal optimization
- **selection_methods.py** — Roulette, tournament, and rank selection comparison
- **crossover_mutation.py** — Crossover and mutation operators on sphere function
- **tsp.py** — Traveling Salesman Problem with order crossover
- **neuroevolution.py** — Evolving a neural network to solve XOR
- **cma_es.py** — Simplified CMA-ES on 2D Rastrigin function

## Run

```bash
python basic_ga.py
python selection_methods.py
python crossover_mutation.py
python tsp.py
python neuroevolution.py
python cma_es.py
```
