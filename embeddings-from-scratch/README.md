# Embeddings from Scratch

Verified, runnable code from the [Embeddings from Scratch](https://dadops.dev/blog/embeddings-from-scratch/) blog post.

## Scripts

- **word2vec.py** — Complete Word2Vec skip-gram implementation with negative sampling, training on a toy corpus, nearest-neighbor queries, and vector arithmetic analogies

## Run

```bash
pip install -r requirements.txt
python word2vec.py
```
