# Hopfield Networks from Scratch

Verified, runnable code from the [Hopfield Networks from Scratch](https://dadops.dev/blog/hopfield-networks-from-scratch/) blog post.

## Scripts

- **hopfield_basic.py** — Binary Hopfield network: Hebbian storage and asynchronous recall
- **hopfield_energy.py** — Energy function tracking during recall
- **hopfield_capacity.py** — Storage capacity experiment (P/N ratio vs accuracy)
- **hopfield_continuous.py** — Modern continuous Hopfield networks with exponential capacity
- **hopfield_attention.py** — Hopfield-attention equivalence proof

## Run

```bash
pip install -r requirements.txt
python hopfield_basic.py
python hopfield_energy.py
python hopfield_capacity.py
python hopfield_continuous.py
python hopfield_attention.py
```
