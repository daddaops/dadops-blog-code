# Logistic Regression from Scratch

Verified, runnable code from the [Logistic Regression from Scratch](https://dadops.dev/blog/logistic-regression-from-scratch/) blog post.

## Scripts

- **logistic_core.py** — Shared module: sigmoid, LogisticRegression class, binary classification demo
- **regularized.py** — L1/L2 regularized logistic regression, sparse feature selection
- **softmax_multiclass.py** — Softmax regression for 3-class classification
- **xor_neural_net.py** — XOR problem: LR fails, 1-hidden-layer network succeeds
- **comparison.py** — Classifier comparison table and weight interpretability

## Run

```bash
pip install -r requirements.txt
python logistic_core.py
python regularized.py
python softmax_multiclass.py
python xor_neural_net.py
python comparison.py
```
